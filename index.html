Our beloved creation - the devops swiss army knife. Simple and beneficial

You can find it useful if you practise Ansible, Vagrant, etc.

devstack

Agile

Opinionated script to quickly create a cloud development environment

Useful

Suitable for development and operational testing

Smart

Demonstrates and documents examples of configuring and running services

DevStack is an awesome tool. It is similar to Ansible - a front end to manage your infrastructure

DevStack's mission is to provide and maintain tools used for the installation of the central cloud services from source

It has evolved to support a large number of configuration options and alternative platforms and support services
